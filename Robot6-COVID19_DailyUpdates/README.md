# Robot6 — COVID-19 Daily Updates

## Description

Scrapes daily COVID-19 statistics from a web source and processes the data for reporting.

## How It Works

1. Navigates to a COVID-19 data website
2. Extracts daily case numbers, deaths, and recovery statistics
3. Processes and formats the data

## Key Concepts

- Web data extraction
- Real-time data scraping
- Data transformation

## Prerequisites

- Internet connection
- Chrome/Edge with UiPath extension

## How to Run

1. Open `project.json` in UiPath Studio
2. Run `Main.xaml`
