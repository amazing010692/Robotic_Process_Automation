# myFirstRobot — Notepad Automation

## Description

An introductory UiPath robot that opens Notepad and writes text, demonstrating basic UI automation.

## How It Works

1. Opens Windows Notepad application
2. Types predefined text into the editor
3. Demonstrates basic UI selectors and type activities

## Dependencies

- UiPath.UIAutomation.Activities
- UiPath.Excel.Activities
- UiPath.PDF.Activities

## How to Run

1. Open `project.json` in UiPath Studio
2. Ensure Notepad is accessible on your system
3. Run `Main.xaml`
