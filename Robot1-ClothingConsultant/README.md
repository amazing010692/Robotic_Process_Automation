# Robot1 — Clothing Consultant

## Description

A decision-based automation that recommends clothing based on user input (e.g., weather/temperature).

## How It Works

1. Prompts the user for weather information
2. Applies decision logic (If/Switch activities)
3. Displays clothing recommendations via message box

## Key Concepts

- Input Dialog activities
- If/Else decision flows
- Message Box output

## How to Run

1. Open `project.json` in UiPath Studio
2. Run `Main.xaml`
3. Respond to the input prompts
