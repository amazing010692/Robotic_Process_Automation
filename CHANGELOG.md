## [1.0.0] - 2024-01-01

### Added

- **Arguments** — Demonstrates UiPath argument passing between workflows
- **myFirstRobot** — Introductory Notepad automation
- **Robot1-ClothingConsultant** — Weather-based clothing recommendation bot
- **Robot2-DataCapturing** — Desktop & web CRM data entry automation
- **Robot3-AutomationChallenge** — Dynamic web form automation (rpachallenge.com)
- **Robot4-RealEstateWebScraping** — Real estate listing web scraper
- **Robot5-PDFDataScraping** — Bulk PDF invoice data extraction (1,000 documents)
- **Robot6-COVID19_DailyUpdates** — COVID-19 daily statistics scraper
